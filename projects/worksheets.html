Coding Worksheets

As someone who is new to coding, I found week 1 very helpful because it got me familiar with the basics. I learned how to set up and publish a GitHub repository, find it in the Finder and on GitHub, and commit my work to save the progress. I also started using Visual Studio Code, which will be my main tool for coding.

I learned how to make a <!DOCTYPE> for a HTML document. I looked at the structure of HTML focusing on block-level elements like <h>, <p>, <ol>, and <ul>, as well as inline elements like <em>, <strong>, and <small>. I also learned how to add links using <a> tag and images using the <img> tag with src and alt attributes.

I looked at the basics of CSS and how to apply it to my HTML documents using the <link> tag. I also explored styling elements such as changing fonts, adjusting space using borders and padding and using flexbox for flexible layouts. Additionally, I learnt how to link multiple HTML files and CSS stylesheets to style different pages of a website.

I focused on using Flexbox to create flexible, responsive layouts. I learned how to style and position menu items, including properties like flex-grow and flex-shrink. I also explored centering images on the page and embedding media, with Youtube videos and audio files. Furthermore, I learned how to pair media items with text and create anchor links.

I learned how to turn <li> elements into buttons and style them with CSS. I also added a footer and created a link button to navigate between pages. I duplicated my first page to create a second page and centered a large horizontal image, before looking at using custom fonts on both Google and Adobe Fonts. Unfortunately, the image didn't display, likely because there were no images in the folder for the worksheet I downloaded however I still understand what needs to be done going forward.
