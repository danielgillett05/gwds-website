Typography Project

Axial System

Axial designs arrange elements along an imaginary line or axis that divides the layout. The arrangement can be symmetrical or asymmetrical, providing structure and visual balance.

Bilateral System

A bilateral system, places all elements symmetrically on either side of a central axis. This creates a formal and balanced design with a clear focal point.

Dilational System

A dilational system arranges elements in a circular pattern radiating outward from a central point. This creates a dynamic and engaging composition often used to convey movement or focus.

Grid System

A grid system, divides the space into evenly spaced sections, creating a structured framework. Elements are placed within these sections for clarity and organisation.

Modular System

The modular system is similiar to the grid system but uses sections of varying sizes and shapes, not following a specific pattern. This allows for more flexibilty and creative placement of elements within the design.

Radial System

A radial system arranges elements to extend outward from a central point of focus, creating a sense of movement and flow. This design approach emphasised the central point, drawing attention towards it.

Random System

In a random system, elements are placed without any specific pattern or order. This creates a spontaneous and dynamic design, often used to convey chaos or energy.

Transitional System

A transitional system arranges elements in layered, informal way, creating a sense of movement or progression. This design approach blends different sizes, shapes, and positions for a more fluid and dynamic composition.

Reflection on Typography Posters

Working on the typography posters was great as it helped me understand how different typography systems shape the look and feel of a poster. I learned how systems like axial, bilateral, and radial create different effects, from clear structure to dynamic movement. It really made me think about how small design choices can change how someone interacts with a layout. Whether it's the order of a grid or the randomness of a chaotic design, I liked how each system had a different vibe and purpose. This project helped me see how important it is to balance the creative side of a poster with clear communication.
